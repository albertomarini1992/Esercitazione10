Attenzione !

Document root temporanea

La document root di questa istanza del web server è memorizzata in /tmp, pertanto il suo contenuto verrà rimosso al prossimo reboot

Lanciare nuovamente il comando apache per fermare il web server

Controllare i file di log per risolvere eventuali malfunzionamenti. Oppure fermare il web server, cancellare la propria cartella dentro in /tmp, e riavviare il web server nella condizione di default.
